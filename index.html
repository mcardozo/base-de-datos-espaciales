Base de datos espaciales

¿Qué es una base de datos espacial?

Es aquella capaz de gestionar objetos geográficos

¿Que factores permiten esto?

  • Los tipos de datos espaciales
  • Funciones espaciales
  • Índices espaciales

Tipos de datos espaciales

Los tipos de datos espaciales se organizan en una jerarquía de tipos

Funciones espaciales

Una función es un conjunto de líneas de código que realizan una tarea específica

Una base de datos espacial proporciona funciones para el análisis de componentes geométricos

  • Relaciones espaciales
  • Manipulación de geometrías
ST_AsText(geography) devuelve text
ST_Distance(geography, geography) devuelve double
ST_AsGeoJson(geography) devuelve text

Índices espaciales

Un índice es un método

Los índice permiten un rápido acceso a un subconjunto de datos

Una base datos estándar se hace generalmente con índices B-tree, no es eficiente para indexar polígonos

¿Qué es PostgreSQL?

  • Potente sistema de gestión de base de datos objeto-relacional (ORDBMS).
  • Liberado bajo una licencia BSD y, por tanto es software libre y de código abierto.
  • Comunidad global de desarrolladores y empresas que lo desarrollan.
  • Diseñado desde el principio con la extensión de tipo en mente

¿Por qué PostgreSQL?

  • Una demostrada fiabilidad e integridad transaccional por defecto (ACID)
  • Cuidadoso soporte para los estándares SQL (completo SQL92)
  • Complementos de extensión de tipos y funciones
  • Un modelo de desarrollo orientado a la Comunidad
  • No hay límite en el tamaño de columna que soportan grandes objetos SIG
  • Estructura de índice genérico (GIST) para permitir índices R-Tree
  • Es fácil añadir funciones personalizadas

¿Qué es PostGIS?

PostGIS convierte el sistema de gestión de bases de datos PostgreSQL en una base de datos espacial.

Postgis

  • Desarrollado por Refractions Research como proyecto de software libre bajo licencia GPL
  • Escrito en lenguaje C, C++ y PL/PgSQL. Utiliza las librerías de software libre GEOS y proj4
  • Incorpora un modelo de topología parecido a la estructura arco-nodo de ArcInfo WorkStation
  • Soporte de datos raster
CREATE EXTENSION postgis;
SELECT postgis_full_version();

Fragments Unleashed

Fragments can be more than just delayed piece of page.

Thanks to some special classes you can add a couple of useful custom behaviours.

  • The .step fragment will partially fades out;
  • The .shy fragment will completely hides itself;
  • You can use this two special fragment types to easily create special effects.

Native Parallax Support

By popular demand Flowtime.js includes native parallax support*.

Simply add a parallax class to anything you want to have parallax enabled and configure the amount of distance for all elements or for a single element using data-parallax attribute.

For an example go to the next page; for more info read the documentation.
